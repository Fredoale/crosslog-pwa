Política de Privacidad

CROSSLOG - Servicios Logísticos | Warehousing
Fecha de última actualización: 25 de octubre de 2025

1. Introducción

Esta Política de Privacidad describe cómo CROSSLOG ("nosotros", "nuestro" o "la empresa") recopila, usa y protege la información personal de los usuarios de nuestra aplicación de gestión de entregas y mensajería de WhatsApp Business.

2. Información que Recopilamos

Recopilamos la siguiente información para el funcionamiento de nuestro servicio:

  • Información del chofer: Nombre completo del chofer asignado a cada HDR (Hoja de Ruta)
  • Información de entregas: Números de remito, direcciones de entrega, fechas y horarios
  • Número de teléfono: Número de WhatsApp del receptor para notificaciones de entregas
  • Información de receptores: Nombre y firma de quien recibe la mercadería
  • Imágenes: Fotografías de remitos firmados para verificación de entregas
  • Texto extraído por OCR: Datos de remitos procesados mediante reconocimiento óptico de caracteres

3. Cómo Usamos la Información

Utilizamos la información recopilada para:

  • Gestionar y rastrear entregas de mercadería
  • Enviar notificaciones de estado de entrega vía WhatsApp Business
  • Generar reportes de entregas completadas
  • Verificar la recepción de mercadería mediante remitos firmados
  • Mantener registros históricos de entregas para nuestros clientes
  • Almacenar PDFs de remitos en Google Drive para archivo y consulta

4. Uso de WhatsApp Business API

Nuestra aplicación utiliza WhatsApp Business Cloud API de Meta para enviar notificaciones sobre el estado de las entregas. Los mensajes enviados incluyen:

  • Confirmación de entrega completada
  • Número de HDR (Hoja de Ruta)
  • Nombre del chofer
  • Número de remito
  • Nombre de quien recibió la mercadería
  • Enlaces a documentos de remitos en Google Drive

Importante: Solo enviamos mensajes a números de teléfono proporcionados por nuestros clientes corporativos para fines de notificación de entregas. No enviamos mensajes de marketing ni spam.

5. Almacenamiento y Seguridad de Datos

Los datos se almacenan de forma segura en:

  • Google Sheets: Registros de entregas y HDRs
  • Google Drive: Imágenes y PDFs de remitos firmados
  • Navegador local: Datos temporales durante el proceso de entrega (localStorage)

Implementamos medidas de seguridad técnicas y organizativas para proteger la información contra acceso no autorizado, alteración, divulgación o destrucción.

6. Compartir Información con Terceros

Compartimos información solo con:

  • Google (Google Sheets y Drive): Para almacenamiento de datos
  • Meta (WhatsApp Business): Para envío de notificaciones
  • N8N (automatización): Para procesamiento de flujos de trabajo
  • Clientes corporativos: Propietarios de la mercadería entregada

No vendemos, alquilamos ni compartimos información personal con terceros para fines de marketing.

7. Retención de Datos

Conservamos la información de entregas durante el tiempo necesario para cumplir con obligaciones contractuales y legales. Los registros históricos se mantienen para auditorías y consultas de nuestros clientes.

8. Derechos de los Usuarios

Los usuarios tienen derecho a:

  • Acceder a su información personal
  • Solicitar corrección de datos inexactos
  • Solicitar eliminación de datos (sujeto a obligaciones legales)
  • Oponerse al procesamiento de datos
  • Retirar el consentimiento en cualquier momento

9. Cookies y Tecnologías Similares

Nuestra aplicación utiliza localStorage del navegador para almacenar temporalmente:

  • Token de autenticación de Google
  • Información del HDR activo
  • Datos de entregas en curso

Esta información se elimina cuando el chofer cierra sesión o completa todas las entregas.

10. Privacidad de Menores

Nuestra aplicación está diseñada para uso empresarial por parte de choferes y personal de logística. No recopilamos intencionalmente información de menores de 18 años.

11. Cambios a Esta Política

Nos reservamos el derecho de actualizar esta política de privacidad en cualquier momento. Los cambios serán notificados mediante actualización de la fecha de "última actualización" en la parte superior de esta página.

12. Cumplimiento con Meta Platform Terms

Esta aplicación cumple con las Condiciones de la Plataforma de Meta (Meta Platform Terms) y las políticas de WhatsApp Business. Específicamente:

  • No solicitamos datos sensibles innecesarios
  • Protegemos la información del usuario de manera adecuada
  • Solo utilizamos WhatsApp Business API para notificaciones transaccionales relacionadas con entregas
  • No compartimos datos de WhatsApp con terceros no autorizados
  • Respetamos las preferencias de privacidad de los usuarios
